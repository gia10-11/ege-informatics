Тип 18 № 27415

Квадрат размером N×N (1 < N < 17). Робот может перемещаться только вправо или вниз. В каждой клетке лежит монета достоинством от 1 до 100. Посетив клетку, робот забирает монету. Определите максимальную и минимальную сумму, которую может собрать робот, пройдя из левой верхней клетки в правую нижнюю.

Пример входных данных (4×4):

1884
10113
13122
2356

Для указанных входных данных ответ: 41 и 22.

📥 Скачать файл: 18 (1).ods
Подробное решение:
Задача решается методом динамического программирования. Создаются две таблицы dp_max и dp_min размером N×N.

Шаг 1: Начальная клетка (0,0) заполняется значением монеты в ней.
Шаг 2: Заполняем первую строку и первый столбец — в них есть только один путь (только вправо или только вниз), поэтому сумма накапливается последовательным сложением.
Шаг 3: Для остальных клеток (i,j) максимальная сумма вычисляется как: dp_max[i][j] = a[i][j] + max(dp_max[i-1][j], dp_max[i][j-1]), так как прийти можно только сверху или слева. Для минимальной суммы используется min вместо max.
Шаг 4: Ответом являются значения в правой нижней клетке dp_max[N-1][N-1] и dp_min[N-1][N-1].
По данным файла получены числа: максимальная сумма = 1204, минимальная = 502. 
with open('18 (1).txt') as f:
    a = [list(map(int, row.split())) for row in f]
n = len(a)
dp_max = [[0]*n for _ in range(n)]
dp_min = [[0]*n for _ in range(n)]
dp_max[0][0] = dp_min[0][0] = a[0][0]
for j in range(1, n):
    dp_max[0][j] = dp_max[0][j-1] + a[0][j]
    dp_min[0][j] = dp_min[0][j-1] + a[0][j]
for i in range(1, n):
    dp_max[i][0] = dp_max[i-1][0] + a[i][0]
    dp_min[i][0] = dp_min[i-1][0] + a[i][0]
for i in range(1, n):
    for j in range(1, n):
        dp_max[i][j] = a[i][j] + max(dp_max[i-1][j], dp_max[i][j-1])
        dp_min[i][j] = a[i][j] + min(dp_min[i-1][j], dp_min[i][j-1])
print(dp_max[n-1][n-1], dp_min[n-1][n-1])  # 1204 502
Ответ: 1204 502 → 1204502

Тип 18 № 29666

Дана последовательность вещественных чисел. Из неё необходимо выбрать несколько подряд идущих чисел так, чтобы каждое следующее было меньше предыдущего. Какую максимальную сумму могут иметь выбранные числа? В ответе запишите только целую часть.

📥 Скачать файл: 18 (12).ods
Подробное решение:
Задача решается одномерной динамикой. Проходим по последовательности, накапливая сумму текущей убывающей подпоследовательности. Если текущий элемент меньше предыдущего — продолжаем накапливать сумму, иначе начинаем новую подпоследовательность с текущего элемента.
По данным файла максимальная сумма = 358,76 → целая часть = 358. 
with open('18_seq.csv') as f:
    a = [float(x) for x in f]
max_sum = cur = a[0]
for i in range(1, len(a)):
    if a[i] < a[i-1]:
        cur += a[i]
    else:
        cur = a[i]
    max_sum = max(max_sum, cur)
print(int(max_sum))  # 358
Ответ: 358

Квадрат 15×15, ладья ходит вправо или вниз на любое количество клеток. Максимальная сумма чисел в клетках, где ладья останавливалась.

📥 Скачать файл: 18 (14).ods
Подробное решение:
В отличие от обычного робота, ладья за один ход может перемещаться на любое количество клеток вправо или вниз. Поэтому для клетки (i,j) максимальная сумма вычисляется как максимальная сумма среди всех клеток в той же строке левее и в том же столбце выше, плюс значение текущей клетки.
Ответ по файлу: 323. 
with open('18_ladya.csv') as f:
    a = [list(map(int, row.split())) for row in f]
n = 15
dp = [[0]*n for _ in range(n)]
dp[0][0] = a[0][0]
for i in range(1, n):
    dp[i][0] = dp[i-1][0] + a[i][0]
    dp[0][i] = dp[0][i-1] + a[0][i]
for i in range(1, n):
    for j in range(1, n):
        max_row = max(dp[i][k] for k in range(j))
        max_col = max(dp[k][j] for k in range(i))
        dp[i][j] = a[i][j] + max(max_row, max_col)
print(dp[14][14])  # 323

Тип 18 № 35992

Робот вправо/вниз. Сумма увеличивается только если число в следующей клетке больше предыдущего. Найти максимальную сумму.

📥 Скачать файл: 18 (16).ods
Подробное решение:
В этой задаче условие добавления числа в сумму зависит от предыдущего шага: число включается в сумму, только если оно больше предыдущего числа на пути. Поэтому в динамическом программировании нужно хранить не только сумму, но и последнее взятое число. Однако можно упростить: при переходе из клетки-предшественника в текущую, если значение в текущей клетке больше значения в предшественнике, то добавляем его к сумме, иначе сумма не меняется.
Результат по файлу: максимальная сумма = 1296. 
with open('18_inc.csv') as f:
    a = [list(map(int, row.split())) for row in f]
n = len(a)
dp = [[0]*n for _ in range(n)]
dp[0][0] = a[0][0]
for i in range(1, n):
    dp[i][0] = dp[i-1][0] + (a[i][0] if a[i][0] > a[i-1][0] else 0)
    dp[0][i] = dp[0][i-1] + (a[0][i] if a[0][i] > a[0][i-1] else 0)
for i in range(1, n):
    for j in range(1, n):
        from_up = dp[i-1][j] + (a[i][j] if a[i][j] > a[i-1][j] else 0)
        from_left = dp[i][j-1] + (a[i][j] if a[i][j] > a[i][j-1] else 0)
        dp[i][j] = max(from_up, from_left)
print(dp[n-1][n-1])  # 1296
Ответ: 1296

Тип 18 № 38952

Робот: начальный запас энергии = число в стартовой клетке, далее если число больше предыдущего — запас увеличивается на это число, иначе уменьшается. Найти максимальный и минимальный запас.

📥 Скачать файл: 18 (20).ods
Подробное решение:
Запас энергии меняется по правилу: при переходе в клетку с бóльшим числом запас увеличивается на это число, при переходе в клетку с меньшим — уменьшается. Начальный запас равен числу в стартовой клетке.
В динамике для каждой клетки нужно рассматривать два варианта прихода (сверху и слева) и вычислять изменение запаса в зависимости от соотношения чисел.
Ответ по файлу: максимальный запас = 3081, минимальный = 1656. 
with open('18_energy.csv') as f:
    a = [list(map(int, row.split())) for row in f]
n = len(a)
e_max = [[0]*n for _ in range(n)]
e_min = [[0]*n for _ in range(n)]
e_max[0][0] = e_min[0][0] = a[0][0]
for j in range(1, n):
    if a[0][j] > a[0][j-1]:
        e_max[0][j] = e_max[0][j-1] + a[0][j]
        e_min[0][j] = e_min[0][j-1] + a[0][j]
    else:
        e_max[0][j] = e_max[0][j-1] - a[0][j]
        e_min[0][j] = e_min[0][j-1] - a[0][j]
for i in range(1, n):
    if a[i][0] > a[i-1][0]:
        e_max[i][0] = e_max[i-1][0] + a[i][0]
        e_min[i][0] = e_min[i-1][0] + a[i][0]
    else:
        e_max[i][0] = e_max[i-1][0] - a[i][0]
        e_min[i][0] = e_min[i-1][0] - a[i][0]
for i in range(1, n):
