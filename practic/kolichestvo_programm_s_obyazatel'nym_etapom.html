Тип 23 № 11358

Условие: Исполнитель А16 преобразует число, записанное на экране.

У исполнителя есть три команды:

1. Прибавить 1
2. Прибавить 2
3. Умножить на 2

Сколько существует программ, которые исходное число 3 преобразуют в число 12 и при этом траектория вычислений содержит число 10?

Решение: Количество программ = f(3→10) × f(10→12).

f(3→10) = 30, f(10→12) = 2. Ответ: 60.

def f(x, y):
    if x > y: return 0
    if x == y: return 1
    return f(x+1, y) + f(x+2, y) + f(x*2, y)
print(f(3,10) * f(10,12))  # 60
