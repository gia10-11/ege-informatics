Задания 21 типа (ЕГЭ информатика): выигрышная стратегия (первым или вторым ходом)

Условия задач: Одна куча камней. Ходы: добавить a или b камней, либо умножить на k. Игра завершается при достижении порога. Нужно найти начальное S, при котором у Вани есть выигрышная стратегия, позволяющая ему выиграть первым или вторым ходом, но нет стратегии выиграть гарантированно первым ходом.

1
Тип 21 № 27804

Условие: Два игрока, Петя и Ваня, играют в следующую игру. Перед игроками лежит куча камней. Игроки ходят по очереди, первый ход делает Петя. За один ход игрок может добавить в кучу один или четыре камня либо увеличить количество камней в куче в пять раз. Игра завершается, когда количество камней в куче становится не менее 68. Победителем считается игрок, сделавший последний ход. В начальный момент в куче было S камней (1 ≤ S ≤ 67).
Найдите минимальное S, при котором одновременно выполняются два условия:
— у Вани есть выигрышная стратегия, позволяющая ему выиграть первым или вторым ходом при любой игре Пети;
— у Вани нет стратегии, которая позволила бы ему гарантированно выиграть первым ходом.

Подробное решение: Петя может первым ходом получить 9, 12 или 40 камней (S+1, S+4, S*5). Если S=8, то после хода Пети возможны: 9, 12, 40. При 40 Ваня выигрывает первым ходом, умножив на 5 (200≥68). При 9 и 12 Ваня добавляет 4 или 1 камень, получая 13. Далее при любом ходе Пети Ваня умножает на 5 и выигрывает вторым ходом. При S<8 Ваня не может гарантированно выиграть первым ходом, а при S>8 Ваня может выиграть первым ходом. Ответ: 8.

def f(x, h):
    if (h == 3 or h == 5) and x >= 68: return 1
    if h == 5 and x < 68: return 0
    if x >= 68 and h < 5: return 0
    if h % 2 == 0:  # ход Вани
        return f(x+1, h+1) or f(x+4, h+1) or f(x*5, h+1)
    else:  # ход Пети (все варианты должны вести к победе Вани)
        return f(x+1, h+1) and f(x+4, h+1) and f(x*5, h+1)

for s in range(1, 68):
    if not f(s, 1) and f(s, 3):  # нет победы за 1 ход, но есть за 1 или 2 хода
        print(s)
        break
# Вывод: 8

Решение: При S=16 Петя может получить: 17, 19, 32. При 32 Ваня удваивает (64≥42) и выигрывает первым ходом. При 17 и 19 Ваня добавляет 3 или 1 камень, получая 20.
