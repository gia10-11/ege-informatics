Тип 15 № 13745 (1 балл)

Для какого наибольшего целого числа А формула

((x ≤ 9) → (x⋅x ≤ A)) ∧ ((y⋅y ≤ A) → (y ≤ 9))

тождественно истинна (принимает значение 1) при любых целых неотрицательных x и y?

Решение:

Преобразуем: (x > 9) ∨ (x² ≤ A) и (y² > A) ∨ (y ≤ 9). Для тождественной истинности необходимо, чтобы A ≥ 9² = 81 и A < 10² = 100. Наибольшее целое A = 99.

Ответ: 99
