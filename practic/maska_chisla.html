Условие: Среди натуральных чисел, не превышающих 109, найдите все числа, соответствующие маске 12345?7?8, делящиеся на 23 без остатка.

В ответе запишите найденные числа в порядке возрастания, справа от каждого числа — результат деления на 23.

Решение: Маска 12345?7?8 означает, что число имеет вид 12345 x 7 y 8, где x и y — цифры (0-9). Перебираем все возможные x и y, проверяем делимость на 23.

Перебор:
123450708 ÷ 23 = 5367426
123451718 ÷ 23 = 5367466
123453788 ÷ 23 = 5367556
123454708 ÷ 23 = 5367596
123456778 ÷ 23 = 5367686
123459768 ÷ 23 = 5367816

Ответ:

ЧислоЧастное (÷23)
1234507985367426
1234517185367466
1234537885367556
1234547085367596
1234567785367686
1234597685367816
from fnmatch import fnmatch
for x in range(0, 10**9, 23):
    if fnmatch(str(x), '12345?7?8'):
        print(x, x//23)

Условие: Найдите натуральные числа, не превышающие 109, соответствующие маске *31*65?, делящиеся на 31 и 2031 (т.е. на НОК), и количество делителей является степенью двойки.

Решение: Число должно делиться на 31·2031 = 62961. Маска: любое количество цифр, затем 31, затем любое количество цифр, затем 65, затем одна цифра.

53831655 26505
333126651 164021
512313657 252247
647931651 319021
831966654 409634

Ответ: 53831655, 333126651, 512313657, 647931651, 831966654

from fnmatch import fnmatch
def count_divisors(n):
    c = 0
    for i in range(1, int(n**0.5)+1):
        if n%i==0: c+=2 if i*i!=n else 1
    return c
for x in range(0, 10**9, 62961):
    if fnmatch(str(x), '*31*65?'):
        if count_divisors(x) & (count_divisors(x)-1) == 0: # степень двойки
            print(x, x//2031)
