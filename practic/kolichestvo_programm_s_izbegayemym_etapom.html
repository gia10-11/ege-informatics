Тип 23 № 13418

Условие: Исполнитель НечетМ преобразует число на экране. У исполнителя НечетМ две команды:

1. Прибавь 1
2. Сделай нечётное (переводит число x в число 2x+1)

Сколько существует программ, которые число 1 преобразуют в число 27, причём траектория вычислений не содержит число 26?

Решение: Используем динамическое программирование. Запрещённое число — 26.

def f(x, y):
    if x > y or x == 26:
        return 0
    if x == y:
        return 1
    return f(x + 1, y) + f(x * 2 + 1, y)
print(f(1, 27))  # 13

Ответ: 13

A. Вычесть 1
B. Умножить на 2
C. Умножить на 3

Сколько существует программ, которые число 3 преобразуют в число 20 и при этом не содержат двух команд A подряд?

Решение: Запрещены две вычитания подряд.

def f(x, y, lastA=False):
    if x > y + 1:
        return 0
    if x == y:
        return 1
    res = 0
    if not lastA:
        res += f(x - 1, y, True)
    res += f(x * 2, y, False)
    res += f(x * 3, y, False)
    return res
print(f(3, 20))  # 4

Ответ: 4

A. Вычесть 1
B. Разделить на 2 (только для чётных)

Решение: Запрещённые числа: 12, 15.

def f(x, y):
    if x < y or x == 12 or x == 15:
        return 0
    if x == y:
        return 1
    res = f(x - 1, y)
    if x % 2 == 0:
        res += f(x // 2, y)
    if x % 3 == 0:
        res += f(x // 3, y)
    return res
print(f(19, 1))  # 43

Решение: Запрещено одновременное появление 20 и 8.

def f(x, y, has20=False, has8=False):
    if x < y or (has20 and has8):
        return 0
    if x == y:
        return 1
    return f(x-1, y, has20 or x-1==20, has8 or x-1==8) + \
           f(x-3, y, has20 or x-3==20, has8 or x-3==8) + \
           f(x//2, y, has20 or x//2==20, has8 or x//2==8)
print(f(31, 3))  # 47315
