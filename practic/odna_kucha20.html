Тип 20 — Выигрышная стратегия (вторым ходом)

Условие для всех задач: Петя не может выиграть первым ходом, но может выиграть своим вторым ходом независимо от того, как будет ходить Ваня.

1 Тип 20 № 27803

Полное условие: Два игрока, Петя и Ваня, играют в следующую игру. Перед игроками лежит куча камней. Игроки ходят по очереди, первый ход делает Петя. За один ход игрок может добавить в кучу один или четыре камня либо увеличить количество камней в куче в пять раз. Например, имея кучу из 15 камней, за один ход можно получить кучу из 16, 19 или 75 камней. У каждого игрока, чтобы делать ходы, есть неограниченное количество камней. Игра завершается в тот момент, когда количество камней в куче становится не менее 68. Победителем считается игрок, сделавший последний ход, то есть первым получивший кучу, в которой будет 68 или больше камней. В начальный момент в куче было S камней; 1 ≤ S ≤ 67.

Найдите два значения S, при которых у Пети есть выигрышная стратегия, причём одновременно выполняются два условия:

— Петя не может выиграть за один ход;

— Петя может выиграть своим вторым ходом независимо от того, как будет ходить Ваня.

Найденные значения запишите в ответе в порядке возрастания без разделительных знаков.

Подробное решение:

1) Проверим, может ли Петя выиграть первым ходом. Для выигрыша за один ход нужно, чтобы после хода Пети в куче стало 68 или больше камней. Это возможно, если S ≥ 14 (прибавив 4), S ≥ 67 (прибавив 1) или S ≥ 14 (умножив на 5: 14×5=70). Таким образом, при S ≥ 14 Петя может выиграть первым ходом. Значит, искомые S должны быть меньше 14.

2) Рассмотрим S = 9. Петя не может выиграть первым ходом (9+4=13, 9×5=45, 9+1=10 — все меньше 68). Его стратегия: добавить 4 камня, получив кучу из 13 камней.

3) Теперь ходит Ваня. Из кучи в 13 камней он может сделать: 14 (+1), 17 (+4) или 65 (×5). Во всех случаях полученное число меньше 68, поэтому Ваня не выигрывает сразу.

4) После хода Вани Петя своим вторым ходом умножает количество камней на 5: 14×5=70, 17×5=85, 65×5=325. Во всех случаях получается число ≥ 68, и Петя выигрывает.

5) Аналогично для S = 12. Петя добавляет 1 камень, получая 13, и далее действует по той же схеме.

Таким образом, S = 9 и S = 12 удовлетворяют условию.

def f(x, h):
    if h == 4 and x >= 68: return 1
    if h == 4: return 0
    if x >= 68: return 0
    if h % 2 != 0:  # Петин ход (ищет выигрышный вариант)
        return f(x+1, h+1) or f(x+4, h+1) or f(x*5, h+1)
    else:            # Ваня ходит (пытается помешать)
        return f(x+1, h+1) and f(x+4, h+1) and f(x*5, h+1)

for s in range(1, 68):
    if not f(s, 1) and f(s, 3):  # не выигрывает за 1 ход, но выигрывает за 3 хода (вторым своим ходом)
        print(s, end=' ')   # 9 12
