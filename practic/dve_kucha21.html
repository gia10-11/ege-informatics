Условие: Два игрока, Петя и Ваня, играют в следующую игру. Перед игроками лежат две кучи камней. Игроки ходят по очереди, первый ход делает Петя. За один ход игрок может добавить в одну из куч (по своему выбору) один камень или увеличить количество камней в куче в два раза. Игра завершается в тот момент, когда суммарное количество камней в кучах становится не менее 77. Победителем считается игрок, сделавший последний ход. В начальный момент в первой куче было семь камней, во второй куче — S камней; 1 ≤ S ≤ 69. Найдите минимальное значение S, при котором одновременно выполняются два условия: у Вани есть выигрышная стратегия, позволяющая ему выиграть первым или вторым ходом при любой игре Пети; у Вани нет стратегии, которая позволит ему гарантированно выиграть первым ходом.

Подробное решение: При S = 30 Петя своим первым ходом может получить одну из четырёх позиций: (7,31), (8,30), (14,30), (7,60). В позициях (7,60) и (14,30) Ваня выигрывает первым ходом (удваивает вторую кучу). Из позиций (8,30) и (7,31) Ваня может получить позицию (8,31) или (14,31). В этом случае после второго хода Пети может возникнуть одна из позиций: (9,31),(8,32),(16,31),(8,62) — во всех случаях Ваня удваивает вторую кучу и выигрывает своим вторым ходом. Таким образом, минимальное S = 30.

def f(x, y, h):
    if (h == 3 or h == 5) and x + y >= 77: return 1
    if h == 5 and x + y < 77: return 0
    if x + y >= 77 and h < 5: return 0
    if h % 2 == 0:
        return f(x+1, y, h+1) or f(x, y+1, h+1) or f(x*2, y, h+1) or f(x, y*2, h+1)
    else:
        return f(x+1, y, h+1) and f(x, y+1, h+1) and f(x*2, y, h+1) and f(x, y*2, h+1)

for s in range(1, 70):
    if f(s, 7, 1):
        print("S =", s)   # 30
        break

Подробное решение: При S=18 Петя может получить (5,18),(16,18),(4,19),(4,72). В позициях (16,18) и (4,72) Ваня выигрывает первым ходом, увеличив соответствующую кучу в 4 раза. Из (5,18) и (4,19) Ваня переводит игру в позицию (5,19). После любого хода Пети (6,19),(20,19),(5,20),(5,76) Ваня утраивает или учетверяет — выигрывает вторым ходом. Меньшие S не дают гарантии.

Условие: За один ход: добавить 1 камень в одну кучу и 2 камня в другую (порядок любой) либо удвоить любую кучу. Сумма ≥ 41. Начальная позиция (8,S), 1≤S≤32. Найдите минимальное S.

Подробное решение: При S=10 возможные ходы Пети: (9,12),(10,11),(16,10),(8,20). (16,10) и (8,20) — Ваня выигрывает первым ходом, удвоив меньшую кучу. Из (9,12) и (10,11) Ваня делает позицию (11,13). После хода Пети (12,13),(13,12),(22,13),(11,26) Ваня выигрывает вторым ходом (удвоением).

def h_game(x,y,t):
    if (t==3 or t==5) and x+y>=41: return 1
    if t==5 and x+y<41: return 0
    if x+y>=41 and t<5: return 0
    moves = [(x+1,y+2),(x+2,y+1),(x*2,y),(x,y*2)]
    if t%2==0: return any(h_game(a,b,t+1) for a,b in moves)
    else: return all(h_game(a,b,t+1) for a,b in moves)
print(min(s for s in range(1,33) if h_game(8,s,1)))  # 10
