Условие: У исполнителя Увеличитель две команды:

1. Прибавь 2.
2. Умножь на 3.

Сколько есть программ, которые число 1 преобразуют в число 31?

Решение: Обе команды увеличивают число. Из 1 нельзя получить чётное число. Пусть R(n) — количество программ из 1 в нечётное n.

Рекуррентные соотношения:

  • Если n не делится на 3, то R(n) = R(n-2)
  • Если n делится на 3, то R(n) = R(n-2) + R(n/3)
R(1)=1
R(3)=R(1)+R(1)=2
R(5)=R(3)=2
R(7)=R(5)=2
R(9)=R(7)+R(3)=2+2=4
R(11)=R(9)=4
R(13)=R(11)=4
R(15)=R(13)+R(5)=4+2=6
R(17)=R(15)=6
R(19)=R(17)=6
R(21)=R(19)+R(7)=6+2=8
R(23)=R(21)=8
R(25)=R(23)=8
R(27)=R(25)+R(9)=8+4=12
R(29)=R(27)=12
R(31)=R(29)=12

Ответ: 12

def f(x, y):
    if x > y: return 0
    if x == y: return 1
    return f(x+2, y) + f(x*3, y)
print(f(1, 31))  # 12

Условие: У исполнителя Прибавитель две команды:

1. Прибавь 1.
2. Увеличь старшую цифру числа на 1.

Сколько есть программ, которые число 35 преобразуют в число 57?

Решение: Команда 2 увеличивает старшую цифру (3→4→5). Можно использовать её не более двух раз.

Случай 1: команда 2 не используется → все команды 1: 1 программа
Случай 2: команда 2 используется один раз → 13 программ
Случай 3: команда 2 используется два раза → 6 программ
Итого: 1 + 13 + 6 = 20

Ответ: 20

def f(x, y):
    if x > y: return 0
    if x == y: return 1
    # команда 1: +1
    res = f(x+1, y)
    # команда 2: увеличить старшую цифру (+10)
    if x % 10 != 9:
        res += f(x+10, y)
    return res
print(f(35, 57))  # 20

Решение: Используем рекурсию с флагом, была ли предыдущая команда умножением.

Подсчёт даёт результат: 77

Ответ: 77

def f(x, y, last_mul=False):
    if x > y: return 0
    if x == y: return 1
    res = f(x+1, y, False) + f(x+2, y, False)
    if not last_mul:
        res += f(x*2, y, True)
    return res
print(f(1, 9, False))  # 77
