Условия задач: Две кучи камней. Ходы: добавить 1 камень в любую кучу или увеличить кучу в k раз (в зависимости от варианта). Игра завершается при достижении суммарного количества камней (или произведения) порогового значения. Победитель — сделавший последний ход. Необходимо найти значения S (начальное количество во второй куче, первая фиксирована), при которых Петя не может выиграть первым ходом, но гарантированно выигрывает своим вторым ходом при любой игре Вани.

Условие: Два игрока, Петя и Ваня, играют в следующую игру. Перед игроками лежат две кучи камней. Игроки ходят по очереди, первый ход делает Петя. За один ход игрок может добавить в одну из куч один камень или увеличить количество камней в куче в два раза. Игра завершается, когда суммарное количество камней в кучах становится не менее 77. Победителем считается игрок, сделавший последний ход. В начальный момент в первой куче было 7 камней, во второй — S камней (1 ≤ S ≤ 69). Найдите два значения S, при которых у Пети есть выигрышная стратегия, причём одновременно: Петя не может выиграть за один ход, но может выиграть своим вторым ходом независимо от того, как будет ходить Ваня.

Подробное решение: Петя не выигрывает первым ходом, но выигрывает вторым. Значит, своим первым ходом он должен создать позицию, из которой любой ход Вани приводит к позиции, где Петя выигрывает следующим ходом.

Рассмотрим S = 34. Петя добавляет 1 в первую кучу: (8,34). Возможные ходы Вани:
- (9,34), (8,35), (16,34), (8,68). Во всех случаях Петя удваивает вторую кучу и получает сумму ≥77.
Рассмотрим S = 31. Петя удваивает первую кучу: (14,31). Ходы Вани: (15,31), (14,32), (28,31), (14,62). Петя удваивает вторую кучу и выигрывает.
При других S Петя либо выигрывает первым ходом, либо не может гарантировать победу вторым. Ответ: 3134.

def f(x, y, h):
    if h == 4 and x + y >= 77: return 1      # Петя выиграл вторым ходом
    if h == 4: return 0
    if x + y >= 77: return 0
    if h % 2 != 0:   # ход Пети (он выбирает выигрышный)
        return f(x+1, y, h+1) or f(x, y+1, h+1) or f(x*2, y, h+1) or f(x, y*2, h+1)
    else:            # ход Вани (все варианты должны вести к победе Пети)
        return f(x+1, y, h+1) and f(x, y+1, h+1) and f(x*2, y, h+1) and f(x, y*2, h+1)

res = [s for s in range(1, 70) if not f(7, s, 1) and f(7, s, 3)]
print("Значения S:", res)  # [31, 34]

Решение: Перебором по специальным правилам найдены значения S от 26 до 43, при которых Петя выигрывает вторым ходом. Таким образом, min=26, max=43.

def g(a, b, p, end):
    if a >= 48 or b >= 48: return p in end
    if p > max(end): return False
    if a > b:
        moves = [g(a+i, b, p+1, end) for i in range(1,4)] + [g(a, b*2, p+1, end)]
    elif a < b:
        moves = [g(a, b+i, p+1, end) for i in range(1,4)] + [g(a*2, b, p+1, end)]
    else:
        moves = [g(a+i, b, p+1, end) for i in range(1,4)] + [g(a, b+i, p+1, end) for i in range(1,4)]
    return any(moves) if ((p+1) % 2 == end[0] % 2) else all(moves)

res = [s for s in range(1, 48) if not g(13, s, 0, [1]) and g(13, s, 0, [3])]
print(f"Min = {min(res)}, Max = {max(res)}")  # 26, 43
