Условие: Исполнитель Фибо преобразует число на экране. У исполнителя есть две команды:

1. Прибавить 1
2. Прибавить 2

Сколько существует программ, которые число 2 преобразуют в число 18, при этом траектория содержит число 9 и не содержит число 14?

Решение: Разобьём задачу на три части: количество программ из 2 в 9, из 9 в 15 (минуя 14), и из 15 в 18.

Пусть R(n) — количество программ из 2 в n. Команды +1 и +2, поэтому R(n) = R(n-1) + R(n-2) при n>3.

R(2) = 1
R(3) = 1
R(4) = R(2)+R(3) = 2
R(5) = R(3)+R(4) = 3
R(6) = R(4)+R(5) = 5
R(7) = R(5)+R(6) = 8
R(8) = R(6)+R(7) = 13
R(9) = R(7)+R(8) = 21

Из 9 в 15 (число 14 запрещено). Считаем P(n) от 9:

P(9)=1
P(10)=1
P(11)=2
P(12)=3
P(13)=5
P(14)=0 (запрещено)
P(15)=P(13)+P(14)=5+0=5

Из 15 в 18:

Q(15)=1
Q(16)=1
Q(17)=2
Q(18)=Q(16)+Q(17)=3

Общее количество: 21 × 5 × 3 = 315

def f(x, y):
    if x > y or x == 14:
        return 0
    if x == y:
        return 1
    return f(x+1, y) + f(x+2, y)
print(f(2,9) * f(9,18))  # 315

Ответ: 315

Решение: Рекуррентность: R(n)=R(n-1)+R(n/2) (если чётное)+R(n/3) (если кратно 3).

R(3)=1
R(4)=R(3)=1
R(5)=R(4)=1
R(6)=R(5)+R(3)=2
R(7)=R(6)=2
R(8)=R(7)+R(4)=3
R(9)=R(8)+R(3)=4
R(10)=R(9)+R(5)=5
R(11)=R(10)=5
R(12)=R(11)+R(6)+R(4)=8
R(13)=R(12)=8
R(14)=R(13)+R(7)=10
R(15)=R(14)+R(5)=11

Из 15 в 50, запрещено 33. Всего программ = 11.

Общее: 11 × 11 = 121

def f(x, y):
    if x > y or x == 33:
        return 0
    if x == y:
        return 1
    res = f(x+1, y)
    if x*2 <= y: res += f(x*2, y)
    if x*3 <= y: res += f(x*3, y)
    return res
print(f(3,15) * f(15,50))  # 121

Условие: Исполнитель Увеличитель преобразует число. Команды:

1. Вычти 2
2. Найди целую часть от деления на 2

При исходном числе 80 результатом является число 1, при этом траектория содержит число 40 и не содержит 20. Сколько таких программ?

Решение: Этап 1: из 80 в 40. Этап 2: из 40 в 1 (минуя 20).

Рекуррентность: F(x)=F(x-2)+F(x//2).

F(80→40) = 1 (только последовательным вычитанием 2)
F(40→1) с запретом 20 = 372

Общее: 1 × 372 = 372

def f(x, y):
    if x < y or x == 20:
        return 0
    if x == y:
        return 1
    return f(x-2, y) + f(x//2, y)
print(f(80,40) * f(40,1))  # 372

Ответ: 372
