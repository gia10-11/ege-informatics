Условие: Текстовый файл состоит из символов X, Y и Z. Определите максимальную длину цепочки вида XYZXYZXYZ... (составленной из фрагментов XYZ, последний фрагмент может быть неполным).

Скачать файл: 24_demo(2).txt

Решение: Используем образец "XYZ". Идём по строке, отслеживая ожидаемый символ (X, затем Y, затем Z, затем снова X). Если символ совпадает с ожидаемым, увеличиваем длину, иначе сбрасываем и начинаем заново с X.

pattern = "XYZ"
cur_len = 0
max_len = 0
pos = 0
for c in s:
    if c == pattern[pos % 3]:
        cur_len += 1
        pos += 1
        max_len = max(max_len, cur_len)
    else:
        if c == 'X':
            cur_len = 1
            pos = 1
        else:
            cur_len = 0
            pos = 0

Тип 24 № 33196

Скачать файл: 24_demo(10).txt

Решение: Проходим по строке, для каждой позиции, где символ равен 'A', увеличиваем счётчик для следующего символа. Затем находим символ с максимальным счётчиком.

Ответ: G

with open('/ege-informatics/practic/files/t24/24_demo(10).txt') as f:
    s = f.readline()
cnt = [0] * 26
for i in range(len(s)-1):
    if s[i] == 'A':
        cnt[ord(s[i+1]) - 65] += 1
max_cnt = max(cnt)
for i in range(26):
    if cnt[i] == max_cnt:
        print(chr(i+65))  # G
        break

Скачать файл: 24_demo(12).txt

Решение: Читаем строки, ищем минимальное количество 'G'. Для выбранной строки считаем частоту букв и находим наиболее частую (при равных — позже по алфавиту).

Ответ: T

min_g = float('inf')
best_line = ''
with open('/ege-informatics/practic/files/t24/24_demo(12).txt') as f:
    for line in f:
        line = line.strip()
        g_count = line.count('G')
        if g_count < min_g:
            min_g = g_count
            best_line = line
cnt = [0] * 26
for c in best_line:
    cnt[ord(c)-65] += 1
max_cnt = max(cnt)
for i in range(25, -1, -1):
    if cnt[i] == max_cnt:
        print(chr(i+65))  # T

Тип 24 № 35998

Условие: Текстовый файл содержит строки различной длины. В строках, содержащих менее 25 букв A, нужно определить максимальное расстояние между одинаковыми буквами в одной строке.

Скачать файл: 24_demo(13).txt

Решение: Для каждой строки с количеством 'A' < 25 находим для каждой буквы разность между последним и первым вхождением, берём максимум.

Ответ: 1004

max_dist = 0
with open('/ege-informatics/practic/files/t24/24_demo(13).txt') as f:
    for line in f:
        line = line.strip()
        if line.count('A') < 25:
            for ch in set(line):
                first = line.find(ch)
                last = line.rfind(ch)
                max_dist = max(max_dist, last - first)
print(max_dist)  # 1004
