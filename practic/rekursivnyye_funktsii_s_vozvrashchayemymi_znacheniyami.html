Тип 16 № 36871 (1 балл)

Алгоритм вычисления значения функции F(n), где n — целое неотрицательное число, задан следующими соотношениями:

F(0) = 0;

F(n) = F(n / 2), если n > 0 и при этом чётно;

F(n) = 1 + F(n − 1), если n нечётно.

Сколько существует таких чисел n, что 1 ≤ n ≤ 1000 и F(n) = 3?

Решение:

Можно доказать, что F(n) — количество единиц в двоичной записи числа n. Для n от 1 до 1023 количество чисел с тремя единицами равно C(10,3)=120. Среди чисел от 1001 до 1023 нет чисел с тремя единицами. Ответ: 120.

Ответ: 120

Чему равно значение выражения F(23) − F(21)?

Решение:

Аналитически находим, что каждая тройка добавляет 2. Количество шагов: от 21 до 2025 с шагом 3: (2025−21)/3 = 668, плюс ещё один шаг для 23. Разность = 2·(668+1) = 1338.

Ответ: 1338
